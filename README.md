🎬 VaiFlix

O VaiFlix é uma aplicação web inspirada no modelo de streaming da Netflix, desenvolvida com o objetivo de simular uma plataforma moderna de consumo de conteúdo audiovisual. O projeto foca na experiência do usuário, oferecendo uma interface intuitiva, responsiva e visualmente atrativa, permitindo navegação por catálogos de filmes e séries de forma dinâmica.

Além do aspecto visual, o VaiFlix também explora conceitos fundamentais de desenvolvimento web, como organização de componentes, consumo de dados, estruturação de layouts e interação com o usuário. A proposta é unir prática e aprendizado, reproduzindo funcionalidades presentes em grandes plataformas de streaming em um ambiente acadêmico e experimental.

Este projeto foi desenvolvido como parte do processo de evolução em desenvolvimento de software, com ênfase em boas práticas, usabilidade e construção de interfaces modernas.
